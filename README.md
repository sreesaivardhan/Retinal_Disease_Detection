# Retinal_Disease_Detection
Compact Multi-Disease Retinal AI System with Hybrid Architecture

## Project Structure
- `data/` - Dataset storage and processing
- `src/` - Source code modules
- `models/` - Trained models and checkpoints
- `results/` - Outputs, metrics, visualizations
- `notebooks/` - Jupyter notebooks for experiments

## Setup
1. Create virtual environment: `python -m venv venv`
2. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Linux/Mac)
3. Install dependencies: `pip install -r requirements.txt`
